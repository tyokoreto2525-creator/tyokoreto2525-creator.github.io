4. Wi-Fi経路の物理分離構成：Raspberry Pi中継活用編

概要

中継装置（Raspberry Pi＋USB Wi-Fiアダプタ×2、および指向性アンテナ）を用いて、公共Wi-Fiへの直接接続を避けた物理的分離通信構成を実現します。 通信端末（ノートPCなど）と中継ノードを物理的に分離することで、「現場に本人が存在しない」状況を作り、防犯カメラや電波ログによる特定リスクを極限まで低減します。

中継装置構成（例）

[ノートパソコン（物理的に距離を置いた安全な場所）] →（SSH接続または物理操作） →[Raspberry Pi（中継ノード）] → USB Wi-Fiアダプタ1（wlan0）：RelayAPとしてPCにWi-Fi提供 → USB Wi-Fiアダプタ2（wlan1）：公共Wi-Fiへのクライアント接続用

wlan0とwlan1はOS設定で役割を固定し、IP転送とNAT設定により中継通信を構築します。

必要な構成機材

1. Raspberry Pi（3B+ 以上推奨）→ 中継ノード本体
項目 推奨ハード / 設定 備考
中継ノード Raspberry Pi 3B+ / 4 microSDに Raspbian Lite を導入（GUI不要）
Wi-Fiアダプタ USB接続 ×2（内1つは高出力モデル） wlan0＝AP用、wlan1＝Wi-Fi接続用
電源 モバイルバッテリー（10,000mAh以上） 稼働時間目安：3〜5時間
通信方式 NAT＋IP転送＋hostapd＋dnsmasq構成 転送ルールは iptables で明示
操作方法 SSH／自動起動スクリプト 設置後は無人運用が理想
2. USB Wi-Fiアダプタ ×2

1つはPCと接続用、もう1つはフリーWi-Fi接続用として使用します。

3. モバイルバッテリー

ラズパイ用電源（数時間稼働を想定）。屋外設置時でも安定稼働できる容量（10,000mAh以上）を推奨します。

4. MicroSD（8GB以上）

OSインストール先として使用します。購入時は店頭・中古店・フリマで現金決済をおすすめします。

設置・中継場所

安全でアクセス可能な物理的な場所（例：ビルの裏、車内、木陰、廃ビル脇など）を選択します。 防犯カメラや目撃者リスクを回避するため、ラズパイを設置後はすぐにその場を離れます。

設置と撤去時にカメラ等に映らないよう注意し、ラズパイには個人ファイルやログを一切残さず、 OS起動時に自動クリア（RAMディスク運用や logrotate 併用）の構成が望ましいです。 事前にモバイルバッテリーや通信範囲のテストを行い、Wi-Fi中継器＋ラズパイ＋指向性アンテナを使用して、 数百メートル先のフリーWi-Fiに接続可能です。

接続方法（運用）

PCを RelayAP に接続（ラズパイがAPモードで提供）し、ラズパイが wlan1 経由でフリーWi-Fiに接続します。 その結果、あなたの通信は「ラズパイ → 公共Wi-Fi」を通ります。

指向性アンテナ（Yagi型・パラボラ型）やアンプ付きUSBアダプタ（例：ALFA AWUS036ACHなど）を活用することで、 視界が開けた環境下では数百メートル先のアクセスポイントにも接続可能です。

まとめ

適切に設定されていれば、ラズパイは1つのWi-Fiアダプタ（wlan1）でフリーWi-Fiに接続し、 もう1つのアダプタ（wlan0）で「あなた専用の中継Wi-Fi（RelayAP）」を提供します。

これにより、パソコンは「RelayAP」に接続することで、ラズパイを通じてフリーWi-Fiに接続する構成が完成します。 この構成は、いわゆる「Wi-Fiリピーター」や「Wi-Fi NATルーター」に近い動作をします。
